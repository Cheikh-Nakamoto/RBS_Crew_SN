## Décisions de design

| Contexte | Règle |
|----------|-------|
| CTA primaire | `solid-premium` (red → gold → green) |
| CTA secondaire | `outline-neon` (gold border) |
| Accent décoratif | `var(--rbs-gold)` |
| Action destructive / branding rouge | `var(--rbs-red)` |
| Contraste mini corps de texte | `text-white/65` (WCAG AA) |
| Contraste helper/muted | `text-white/55` (WCAG AA large) |
| Touch target | 44×44 px min (glow hover gold) |
| Grayscale images cards | `[0.2-0.3]` → `0` on hover (700ms) |
| Focus ring | `var(--rbs-gold)/70` + offset-4 ring-offset-black |
| Numérotation éditoriale | mono badge `black/60 backdrop-blur-md` top-left |
| Brand stripe card | gold → red → transparent (vertical w-1 h-12-14 top-right) |
